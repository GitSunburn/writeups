Random Security - {{ RS_SERVER }}

Welcome on one of the servers of Random Security. To interact with this server you can use the API. For some features you will have to provide your security key.

/api/help
Display this help page.
/api/get_time
Display the current time of the server.
/api/last_accessed_server
Display the server name of which the secret was most recently accessed.
/api/last_access?key=<key>
Display the last time the secret on this server was accessed.
/api/get_secret?key=<key>
Display the sectret of this server.
